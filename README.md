## Projects

### MigraCRM
Full-stack CRM platform for immigration law call centers.

- **Backend:** Python, FastAPI, PostgreSQL, SQLAlchemy
- **Frontend:** React 18, TypeScript, Vite, Tailwind CSS
- **AI:** Google Gemini 2.5 Flash + faster-whisper (local Whisper transcription)
- **Telephony:** Grandstream UCM6304 PBX integration via CDR/recording APIs
- **Bot:** Hermes Telegram AI agent (Ollama llama3.2 local inference)
- **Deploy:** Bare-metal PM2 on AlmaLinux VPS

Features: automatic call transcription, AI summaries & sentiment analysis, agent notes, caller history, CDR records, live transcriptor status.

### Callio Wave Add-in
Browser extension for Grandstream Wave softphone that shows real-time AI call summaries to agents as calls complete.

- Built with vanilla JS + WebExtensions API
- Polls MigraCRM backend for processed summaries
- Displays caller history, AI notes, action items
